PAK file editors, be warned

We've been randomly checking PAK files for people who've edited their game files. While we're pleased to say that there wasn't been there has sadly still been some. Server roll backs = Duped items. Trade at your own risk

The recent crashing of the servers has resulted in duplicated items (dupes). We're currently in the process of looking through the logs in order to remove them.

We're not accusing players of duping the items, simply removing the one that shouldn't be in game.

Dupe items are created when the server hiccups and isn't able to save the data. When the server is rebooted, the most recent copy of your character is restored. If you've transferred or traded items to another user the restoration can create a duplicate item, this is abuse and against the terms and conditions. By not notifying iEntertainment you are putting yourself at risk. If you're unsure whether or not an item has been duped, contact a Game Master.

Items that have spawned from the 1 user are being removed from this person. We believe that they've knowingly passed the item while holding one with the intent to hold the other for safety.

Items that have rolled back to another user are being removed from who we believe to be the owner of the item. This is at Administrations discretion. If you disagree with the decision please contact us via email.

If we can see an exchange for the item, this has been undone. If the items were traded, but are not logged as being an "exchange" we are unable to assist you since we can not confirm this as a trade.

Please be aware that knowingly trading the items off will result in the item not being returned to you.
Attempting to hide or any other strange behavior with an item will
be viewed as suspicious and could result in further punishment.

Emails are being sent for each item that's been removed.

We aim for this to be solved as quickly as possible. We recommend that you do not trade for any large items at this stage. If you wish to check the validity of an item you're trading for please create a MISC TIXX and we can verify this for you.

If you find that you've got a duped item on your character, please sell 1 (doesn't matter which) to a shop keeper. If you're unsure, please create a TIXX. Be sure to keep both copies on your character (Warehouse the copy). Do not pass the item around or trade it away as this could be viewed as intentional abuse and result in punishment.

Should you have any further questions on the issue please create a MISC TIXX within the Helsupport system.
